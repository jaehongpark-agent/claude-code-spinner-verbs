# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This project extracts the playful spinner verbs (gerund-form loading messages like "Cogitating", "Flibbertigibbeting") from the Claude Code CLI binary and tracks them across versions. It also provides Korean translations and a customization guide for users who want their own spinner verbs.

## Commands

```bash
# Extract spinner verbs from the installed Claude Code binary
python3 extract_spinner_verbs.py

# Extract + regenerate llms.txt (the AI-agent-facing context file)
python3 build.py
```

No dependencies beyond Python 3.10+ stdlib. Requires Claude Code CLI installed and the `strings` command (pre-installed on macOS/Linux).

## Architecture

**Two-script pipeline:**

1. `extract_spinner_verbs.py` — Standalone extractor. Finds the Claude Code binary (`which claude` or `~/.local/share/claude/versions/`), runs `strings` on it, uses seed-based bootstrapping with known rare words (e.g. "Flibbertigibbeting", "Razzmatazzing") to locate the spinner array in the binary output, then parses all gerund-form words via regex. Returns `(version, words)` tuple.

2. `build.py` — Imports `extract_spinner_verbs.main()`, then generates `llms.txt` from the extracted words plus optional Korean translations.

**Key constants in `extract_spinner_verbs.py`:**
- `BOOTSTRAP_SEEDS` — 12 rare words used to locate the spinner array in binary output
- `SEED_MATCH_THRESHOLD` (3) — minimum seed matches to consider a line as the spinner array
- `GERUND_RE` — regex matching capitalized words ending in `-ing` or `-in'` (including accented chars)

**Data files in `words/`:**
- `<version>.md` — one word per line, sorted alphabetically
- `<version>_kr.md` — `English: 한국어` pairs per line (manually created)

**`llms.txt`** — Auto-generated by `build.py`. Do not edit manually.

## Conventions

- The project uses the term "spinner verbs" (not "spinner words") consistently throughout
- Word lists are plain markdown with one word per line, no bullet points or numbering
- Korean translation files use the `_kr` suffix (e.g., `2.1.63_kr.md`)
- `CUSTOMIZE.md` / `CUSTOMIZE_KR.md` document the `spinnerVerbs` setting in `~/.claude/settings.json`
